# Web Blue Fleet

## Sobre o projeto

Aplicação de gestão de veículos que permita listar, cadastrar, alterar e excluir veículos.

## Definições de negócio

* O veículo deve ter placa, montadora e modelo
* Listar veículos
* Cadastrar veículo
* Alterar veículo
* Excluir veículo
